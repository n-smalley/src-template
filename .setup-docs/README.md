# Repo / Folder Setup Instructions
## Prerequisites
- Ensure `.setup-docs\requirements.txt` is up to date with all required dependencies

## Environment Setup
### 1. Open a New Terminal
```
Ctrl+Shift+`
```
### 2. Run the Virtual Environment Setup Script
Execute the batch file:
```bash
.setup-docs\.venv_setup.bat
```
- Wait for setup to complete
- Python version should be `3.12.10`

## Notes
If changes are made to `.setup-docs\requirements.txt`, rerun `.setup-docs\.venv_setup.bat` in terminal to download new modules
